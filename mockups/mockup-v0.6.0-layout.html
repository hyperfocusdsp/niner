DStacked compact LCD selectors + small comp-cluster knobs. Top sub-row: SAT MODE (compact) + SAT DRV + SAT MIX + empty. Bottom sub-row: CLIP MODE (compact, displaying tAnH / dIOdE / CUbIC) + CLIP DRV + N DECAY + ACCENT. EQ unchanged on the right. SAT/EQ row grows by ~22px to fit two stacked sub-rows; STEP grid + DICE shift down. CLIP MODE labels use lowercase n in tAnH because the 7-segment glyph table doesn't define uppercase N.

WHY

Visual symmetry between SAT and CLIP — they're sibling stages (master-bus saturation vs per-voice waveshaper). Compact dimensions free up enough horizontal room to fit the four new controls without crowding. Re-uses the existing comp-cluster style so nothing visually invented.

RISKS COVERED

7-seg glyph coverage verified for all CLIP labels. lcd_selector refactor adds id_source param to avoid egui Id collision. Captions stay below knobs (current convention). Window has 74px slack so the 22px row extension fits at all UI scales.
